About HTML

The history of HTML :

Tim Berners-Lee, a physicist at CERN, invented HTML in 1991.
The first version of HTML had 18 tags,but there are now around 140 .
HTML 4.01 was the most popular version in the 2000s, and XHTML was a rewrite of HTML as an XML language.
HTML5, released in 2014, is the current standard for web development.

How HTML is used ?

HTML is the backbone of the internet and is used to create web pages and applications.
When combined with CSS and JavaScript, HTML is a key part of web development.
CSS helps with layout, colors, and appearance, while JavaScript helps with embedding programs and affecting the behavior of web page contents
